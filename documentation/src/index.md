# Tarnished documentation

Tarnished is a self-hosted job application tracker for managing applications, job leads, documents, interview rounds, analytics, imports, exports, and related automation.

This documentation site is organized around clear user needs:

- **Getting started** for first-time setup and orientation
- **Tutorials** for guided learning
- **How-to guides** for task-focused work
- **Reference** for factual technical details
- **Explanation** for architecture and concepts
- **Troubleshooting** for diagnosing and fixing problems

## Start here

- New to Tarnished? Start with **Getting started**.
- Need to complete a specific task? Go to **How-to guides**.
- Need exact technical details? Go to **Reference**.
- Need to understand how something works? Go to **Explanation**.
