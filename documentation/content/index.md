Tarnished is a self-hosted job application tracker for managing applications, job leads, documents, interview rounds, analytics, imports, exports, and automation on your own infrastructure.

## Start here

### Install Tarnished

Choose the install path that matches your environment:

- [Install with Docker Compose](./install/docker-compose.md) — the fastest way to get Tarnished running
- [Install with PostgreSQL Docker Compose](./install/postgresql-docker-compose.md) — start with PostgreSQL from day one
- [Install with Helm](./install/helm.md) — run Tarnished on Kubernetes
