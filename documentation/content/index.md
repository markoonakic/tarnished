Tarnished is a self-hosted job application tracker for people who want to manage applications, job leads, documents, interview rounds, analytics, imports, exports, and related automation on their own infrastructure.

This site is the main documentation entry point for Tarnished.

## What Tarnished helps you do

With Tarnished, you can:

- track applications across custom statuses and interview rounds
- save and convert job leads from the browser extension
- store CVs, cover letters, transcripts, and media files
- import and export your data for migration and backup workflows
- run analytics and AI-assisted insights against your job-search pipeline
- manage API keys for the CLI, extension, and other machine clients

## Documentation structure

Tarnished documentation follows a Diátaxis-style structure:

- **Getting started** for first-time setup and orientation
- **Tutorials** for guided learning
- **How-to guides** for task-focused work
- **Reference** for factual technical details
- **Explanation** for architecture and concepts
- **Troubleshooting** for diagnosing and fixing problems
- **Contributing** for docs and contributor guidance
