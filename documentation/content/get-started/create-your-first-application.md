This guide introduces the core Tarnished workflow by walking through a first application entry.

## Before you begin

You need:

- a working Tarnished install
- a Tarnished user account

If you have not created the first account yet, start with [Create your admin account](./create-admin-account.md).

## Step 1: Create an application

Create a new application entry with the essential information:

- company
- job title
- applied date
- status
- optional job URL or job description

## Step 2: Review it in the applications list

After saving the application, confirm that it appears in the list.

At this point Tarnished has started tracking the application as part of your pipeline.

## Step 3: Update the status over time

As the application progresses, update the status.

Tarnished records the journey of an application across statuses, which later powers the pipeline, dashboard, and analytics views.

## Step 4: Add rounds and supporting files

As interviews happen, add rounds and attach supporting files as needed.

Typical examples include:

- CV
- cover letter
- transcript
- interview audio or video
