Use this section to get Tarnished running and choose the right install path for your environment.

## Choose an install path

### You want the fastest local start

Use [Quickstart with Docker Compose](./quickstart-docker-compose.md).

This is the simplest path for trying Tarnished quickly with the default SQLite-backed install.

### You want PostgreSQL from the start

Use [Deploy with PostgreSQL Compose](./deploy-with-postgresql-compose.md).

This is the better path if you want a local PostgreSQL container instead of the default SQLite setup.
